# Домашнее задание №1. Программирование на Python

Работа выполнялась в Google Colab. В файле находятся решения двух задач из задания.

## Задача 1

Программа запрашивает у пользователя слово и выводит:
- одну среднюю букву, если длина слова нечётная,
- две средние буквы, если длина слова чётная.

Примеры:
test → es  
testing → t  

Для решения использовались функции input(), len(), деление //, %, а также индексация строки.

## Задача 2

Во второй задаче нужно составить пары из двух списков: мальчиков и девочек.  
Работа была выполнена в двух вариантах, как указано в условии.

### Вариант 1. Длины списков совпадают

В списках по 5 имён.  
Списки сортируются методом sort(), после чего программа выводит пары по индексам.

Пример результата:

Alex и Emma  
Arthur и Kate  
John и Kira  
Peter и Liza  
Richard и Trisha

В список мальчиков добавлено дополнительное имя, из-за чего длины списков стали разными.

В решении использовались: sort(), len(), цикл for и условные операторы.
