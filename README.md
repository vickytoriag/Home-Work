### ДЗ 7. Работа с API

**JSONPlaceholder.py**  
Скрипт выполняет GET-запрос к публичному API JSONPlaceholder
(`/posts`) и выводит заголовки и тела первых 5 постов.

**OpenWeather.py**  
Скрипт принимает название города от пользователя и выполняет
GET-запрос к API OpenWeather для получения текущей температуры
и описания погоды.


Для работы используется API-ключ, который хранится в файле `.env`
и не публикуется в репозитории.
