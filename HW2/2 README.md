# Домашнее задание №2

Работа выполнялась в Google Colab. В задании нужно было реализовать две функции.

## Задача 1. sum_distance

Функция суммирует все целые числа от первого аргумента до второго включительно.  
Если числа переданы в неправильном порядке, они меняются местами.

## Задача 2. trim_and_repeat

Функция принимает строку, количество символов для обрезки слева и число повторений.  
Параметры offset и repetitions имеют значения по умолчанию: 0 и 1 соответственно.  
После обрезки строка повторяется нужное количество раз и возвращается.
